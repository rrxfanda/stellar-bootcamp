# Stellar Tip Jar (Soroban dApp)

A decentralized tip jar smart contract built on the Stellar Soroban platform. This project allows users to send tips, track total contributions, and store tipping history directly on-chain. It includes features such as user-based statistics, creator earnings tracking, and message support for each tip. Designed as a backend-only dApp, it runs entirely in Soroban Studio without requiring a frontend.
